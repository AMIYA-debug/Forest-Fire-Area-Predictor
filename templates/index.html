Algerian Forest Fire Area Predictor

This project leverages machine learning techniques to forecast the area impacted by forest fires in Algeria. It relies on meteorological and environmental parameters such as temperature, relative humidity, wind speed, rainfall, and several fire-related indices.

The underlying model has been trained on a real-world dataset collected from Algeriaâ€™s Bejaia and Sidi-Bel Abbes regions, recorded over the summer months (June to September). The dataset includes fire occurrences alongside weather measurements, enabling effective prediction of fire-affected zones.

By applying a Multiple Linear Regression approach, the model predicts the likely burned area (in hectares) based on input features. This tool can assist environmental authorities, disaster response teams, and researchers in enhancing early warning systems, optimizing response strategies, and understanding wildfire behavior.
